## STEP 1: GitHubアカウントの確認・作成

STEP 2 に進んでください。

---

### GitHubアカウントをお持ちでない方

個人用のGitHubアカウントを作成します。  
GitHubは無料で利用できます。

#### 作成手順

1. ブラウザで **`https://github.com`** を開く

2. 「Sign up」ボタンをクリック

3. 以下の情報を入力する：

   | 項目 | 入力内容 |
   |------|---------|
   | メールアドレス | 受け取れるメールアドレス |
   | パスワード | 英数字・記号を含む8文字以上 |
   | ユーザー名 | 半角英数字・ハイフンのみ使用可（後から変更可能） |

4. ロボット認証（パズル）を完了する

5. 登録したメールアドレスに届いた確認コード（6桁）を入力する

6. プラン選択画面が表示されたら **「Free」** を選択する

7. GitHubのホーム画面が表示されれば作成完了

#### ユーザー名について

- 半角英数字とハイフン（`-`）が使えます
- 大文字・小文字は区別されません
- 後から変更できますが、変更すると関連するリンクが切れる場合があります
- **例：** `yamada-taro`、`YamadaTaro`、`y-takamura`

> **注意：** GitHubはインターネット上の公開サービスです。ユーザー名に個人情報が過度に含まれないよう注意してください。

## STEP 2: Git Bashのインストール（Windowsのみ）

Git Bashは、Windows上でGitコマンドを実行するためのターミナルです。

### インストール済みか確認する

1. スタートメニューで「Git Bash」を検索する
2. 「Git Bash」が見つかればインストール済みです → **STEP 3 に進む**
3. 見つからない場合は以下の手順でインストールします

### インストール手順

1. ブラウザで **`https://git-scm.com/downloads`** を開く

2. 「Download for Windows」をクリック  
   （ページを開くと自動でダウンロードが始まる場合があります）

3. ダウンロードされたインストーラーをダブルクリックして実行  
   （ファイル名例：`Git-2.47.1-64-bit.exe`）

4. 「ユーザーアカウント制御」のダイアログが出たら「はい」をクリック

5. インストールウィザードが起動します。**すべてデフォルト（初期設定）のまま「Next」を押し続けてください**

   > 設定を変更する必要はありません。わからない場合はすべてNextでOKです。

6. 「Install」をクリックしてインストール開始

7. 「Finish」をクリックして完了

### インストールの確認

1. スタートメニューで「Git Bash」を検索してクリック
2. 黒いウィンドウ（ターミナル）が開く
3. 以下のコマンドを入力して **Enter** を押す：

```bash
git --version
```

4. 以下のように表示されればOKです：

```
git version 2.47.1.windows.1
```

（バージョン番号は多少異なっていても問題ありません）

## STEP 3: Gitの初期設定

GitHubアカウントとGitを紐づけるための設定です。**必ず実施してください。**

### 設定手順

1. **Git Bash** を開く（スタートメニューから起動）

2. 以下のコマンドを1行ずつ入力し、**Enter** を押す

```bash
git config --global user.name "【GitHubのユーザー名】"
```

```bash
git config --global user.email "【GitHubに登録したメールアドレス】"
```

**入力例：**
```bash
git config --global user.name "yamada-taro"
git config --global user.email "yamada.taro@example.com"
```

> `"` ダブルクォーテーションで囲むことを忘れずに。

3. 設定内容を確認する：

```bash
git config --global --list
```

以下のように表示されればOKです：

```
user.name=yamada-taro
user.email=yamada.taro@example.com
```

## よくある質問（事前準備）

**Q: GitはPIパソコンで利用できますか**  
A: PIパソコンのセキュリティ対策のため、PIパソコンではGitコマンドは利用できません。開発用パソコンでご利用ください。

**Q: git --version を実行したらエラーになりました**  
A: Git Bashを再起動してから再試行してください。それでも解決しない場合は再インストールをお試しください。

**Q: user.name に日本語（漢字）を使っても大丈夫ですか？**  
A: GitHubのユーザー名は半角英数字のみです。`git config --global user.name` にはGitHubのユーザー名（半角英数字）を入力してください。

**Q: GitHubアカウントの作成でメール確認コードが届きません**  
A: 迷惑メールフォルダを確認してください。または別のメールアドレスで再試行してください。
